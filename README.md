# Mobile Hero

Landing page built with **Next.js 16**, **Tailwind CSS**, and **TypeScript**.

---

# Tech Stack

* Next.js 16
* React
* TypeScript
* Tailwind CSS

---

# Project Setup

## 1. Clone Repository

```bash
git clone https://github.com/krishnaa192/Mobile-Hero.git
```

---

## 2. Navigate Into Project

```bash
cd Mobile-Hero
```

---

## 3. Install Dependencies

```bash
npm install
```

---

## 4. Start Development Server

```bash
npm run dev
```

Application runs on:

```bash
http://localhost:3000
```

---

# Production Build

## Create Production Build

```bash
npm run build
```

---

## Run Production Server

```bash
npm start
```

---


# Tailwind Setup

Tailwind CSS is configured using:

* postcss.config.mjs
* tailwind.config.js
* global.css

---

| Command       | Description              |
